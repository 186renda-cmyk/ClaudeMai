一、 MCP 到底是什么？

MCP（Model Context Protocol）是 Anthropic 推出的一套开放标准协议。在没有 MCP 之前，如果你想让 AI 连接一个数据库，你可能需要写大量的胶水代码。而有了 MCP，这就变成了一个“即插即用”的过程。

🧠 MCP vs Skills：手 vs 脑

很多人容易混淆这两个概念。
MCP（手）： 提供“干活的能力”。比如“读取文件”、“发送请求”，它是底层的硬件基础。
Skills（脑）： 提供“怎么做”的流程。比如“如何分析一份财报”，它是上层的思维逻辑。

二、 MCP 能做什么？

安装了不同的 MCP 服务（Server）后，Claude 就能解锁各种新技能：

  • 操作本地文件 (Filesystem MCP)： 让 Claude 直接读取、修改、重命名你电脑里的文件，无需复制粘贴。
  • 联网搜索 (Brave/DuckDuckGo MCP)： 赋予 Claude 实时联网能力，查询最新资讯和技术文档。
  • 管理代码 (GitHub MCP)： 直接审查 Pull Request，查看仓库结构，甚至创建 Issue。
  • 浏览器自动化 (Playwright MCP)： 像真人一样操作浏览器，进行网页截图、数据抓取或自动化测试。

三、 如何配置与使用？

要体验 MCP，你需要安装 Claude 桌面版应用。配置过程其实非常简单，主要分为三步：

1. 准备环境

确保你的电脑上安装了 Node.js（版本建议 ≥ 18）。你可以通过终端输入 node -v 来检查。

2. 使用命令行安装 (推荐)

Anthropic 提供了便捷的命令行工具。例如，要安装 GitHub 支持，只需运行：

claude mcp add --transport http github https://api.github.com/mcp

3. 手动配置文件 (进阶)

你也可以直接编辑 Claude 的配置文件（通常位于 ~/Library/Application Support/Claude/claude_desktop_config.json ），添加你需要的服务：

{ "mcpServers": { "filesystem": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/username/Desktop"] } } }

配置完成后，重启 Claude 桌面端，你会发现输入框旁边多了一个“工具”图标，这就说明 MCP 已经生效了！

四、 实用场景举例

场景一：全自动代码重构

结合 Filesystem MCP，你可以直接对 Claude 说：“帮我把 src 目录下所有的 CSS 文件重构为 Tailwind 类名”。Claude 会自动读取文件、修改代码并保存，你只需要在一旁喝咖啡。

场景二：智能研报生成

结合 Search MCP，你可以问：“帮我调研一下 2026 年 AI 编程工具的市场份额，并生成一份 Markdown 报告”。Claude 会自动联网搜索最新数据，然后整理成文。

总结： MCP 是 AI 进化的重要一步，它让大模型从“单机版”变成了“联网版”，从“对话者”变成了“执行者”。如果你想最大化 Claude 的生产力，尽早掌握 MCP 的配置绝对是明智之举。
